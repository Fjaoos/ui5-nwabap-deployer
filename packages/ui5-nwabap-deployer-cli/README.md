# ui5-task-nwabap-deployer

`ui5-nwabap-deployer-cli` is a CLI tooling which allows to deploy UI5 sources to a SAP NetWeaver ABAP application server.

## Install
```bash
npm install ui5-task-nwabap-deployer-cli --save-dev
```
